# News Images

Put article images here using one folder per news slug.

Example:
- `images/main-site-notifications-launch/thumb.jpg`
- `images/main-site-notifications-launch/cover.jpg`

For JSON records, reference them with stable site paths, for example:
- `/news-content/images/main-site-notifications-launch/thumb.jpg`
- `/news-content/images/main-site-notifications-launch/cover.jpg`

If a news item has no image yet, leave `thumbnail_url` and `cover_image_url` empty. The frontend will render a placeholder card.
