This page automatically publishes the server leaderboard including:

  • 🏁 number of races
  • 🥇 wins
  • 🏆 podium finishes
  • 📊 average finish position
  • ⚡ best laps

Statistics are generated automatically from ACC Dedicated Server result files. After each race the data is recalculated and published on the website.

How points are calculated

Race points depend on how many drivers are classified in that race:

  • 25+ drivers - base GT scale 25/18/15/12/10/8/6/4/2/1
  • 20-24 / 15-19 / 10-14 drivers - the same scale is reduced so the winner gets 20 / 15 / 10 points
  • 5-9 drivers - the winner gets 5 points, with the rest of the scale reduced proportionally
  • 1-4 drivers - the winner gets points equal to the number of classified drivers

All scaled values are rounded to whole points: 0.1-0.5 rounds down, 0.6-0.9 rounds up. The fastest lap always gives +1 point.

Best laps

The Best Laps table contains the fastest lap times recorded both in qualifying and in race sessions. This makes it easy to compare the outright pace of the drivers.
