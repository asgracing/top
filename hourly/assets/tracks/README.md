Place track hero assets in this folder using these exact filenames:

- `monza.jpg`
- `monza-outline.png`
- `silverstone.jpg`
- `silverstone-outline.png`
- `spa.jpg`
- `spa-outline.png`
- `nurburgring.jpg`
- `nurburgring-outline.png`

The page will use the photo as the primary hero backdrop and the outline as the circuit badge in the announcement card.
If a file is missing, the site falls back to the generated vector art already built into `app.js`.
